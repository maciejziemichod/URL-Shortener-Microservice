URL Shortener Microservice

Features:

  1. You can POST a URL to [project_url]/api/shorturl/new and API will response with a JSON containing shortened URL.
    Example : {"original_url":"www.google.com","short_url":1}
  2. If URL doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error {"error":"invalid URL"}

Short URL Creation

example: POST /api/shorturl/new - https://www.google.com

Example Usage:

/api/shorturl/1

Will Redirect to:

https://www.freecodecamp.org
